  • Mapping

    A MockFloret.Mapping is an internal representation of a mapping from a URLRequest to a Result for a Response ((URLRequest, MockFloret) -> Result<Response>?). This Mapping will be returned when using the addMapping function and can be used to remove a Mapping via removeMapping.
